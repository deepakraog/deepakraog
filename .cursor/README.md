## 🔧 Setting Up Automations

1. Review templates in `AUTOMATION_CONFIG_TEMPLATE.md`
2. Go to [cursor.com/automations](https://cursor.com/automations)
3. Click "New Automation"
4. Copy desired template
5. Configure trigger and repositories
6. Save and activate

## 📊 Environment Configuration

For persistent environment setup across cloud agent runs, configure at:
- [cursor.com/onboard](https://cursor.com/onboard)

This allows you to:
- Define base images
- Install common dependencies
- Configure startup scripts
- Set environment variables

## 🔐 Secrets Management

Add secrets for agents at:
- [cursor.com/cloud-agents/secrets](https://cursor.com/cloud-agents/secrets)

Secrets are automatically injected as environment variables in agent runs.
